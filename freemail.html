Step 3: Creating a mail account with Freemail

You must have already created a Web of Trust (WoT) identity in step 2 of this guide before proceeding.

Plugin setup

  1. Start by opening the Plugin configuration page.

  2. The version of Freemail used in this guide is newer than the official plugin included with Freenet 1407 so we need to install it as if it were an unofficial plugin.

    Copy the following text into the Plugin Key field and click 'Load'. 

    CHK@lIazySXUwRohUUcYD5ynh8IVA42SaiVLPsgHPHtk5Uc,3LKam0sxFEyZzdsiFlnkBpXPL0h1BPNCTjDbLQ~y9dw,AAIC--8/Freemail-0.2.2.jar

  3. When the plugin is loaded log in to Freemail using the new top-level menu.

  4. Since you don't have any accounts yet you will be prompted to add one.

  5. If you have more than WoT identity choose the one you want to create a Freemail account for. Choose a password and click 'Add account'. Don't forget the password because you'll need it later.

  6. Now that you've created an identity you can log in.

  7. This is your inbox. The first message contains useful information, including you new Freemail address.

  8. The Freemail plugin contains a basic web interface that allows you to send and receive messages. If you want you can use this interface but for full functionality you can set up an external mail client. Continue with this guide for instructions on setting up Thunderbird.

Email client setup

  1. If this is the first time you've used Thunderbird it will immediately prompt you to set up an account. If you already have accounts go to the 'Account Settings' menu. Under the 'Account Actions' menu choose "Add Mail Account...'

    Click past this prompt to configure Thunderbird to access your Freemail account.

  2. Choose a name to display on your outgoing emails and enter the email addressed contained in your "Welcome to Freemail" message. Use the same password here as you choose in step of 5 the previous section and click 'Continue'.

  3. Thunderbird will try to automatically detect your mail server settings but it won't work.

  4. Change the Server hostname, Port and Username fields as shown below and click 'Re-test'.

  5. If you entered everything correctly Thunderbird will successfully detect the rest of the parameters. Click on 'Create Account' to continue.

  6. This warning occurs because normally it's a bad idea to connect to a mail server over an insecure connection. Since this connection takes place on the localhost interface TLS is not necessary. Click on the checkbox and click 'Create Account' again.

  7. At this point you should see the Welcome email from your inbox. From now on you can do everything relating to your mail from Thunderbird just like any other email service. Just remember that your node must be running to use your Freemail account.

  8. If you ever plan to send messages with non-English characters there is a setting you should change to make sure that your messages display properly.

    Go to the Thunderbird configuration menu.

  9. Under the Advanced tab choose 'Config Editor'.

  10. Acknowledge the warning. If you wish, you can uncheck 'Show this warning next time'.

  11. Search for "mail.strictly_mime" and double click to change the value to 'true'.
